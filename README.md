# Java-Scenario-Game
A console-based Java scenario game that simulates a first-year university student's first week on campus. Players make daily decisions that affect their energy levels and academic performance, demonstrating object-oriented programming concepts and user input validation.

## Project Description
This project is a console-based Java application developed as part of my Higher Certificate in Information Systems (Software Engineering) studies.

The game simulates a first-year university student's first week on campus. Players make daily decisions that influence their energy levels and academic performance, helping them understand how choices can affect success at university.

## Project Features
-Student name validation using regular expressions
-Object-oriented programming implementation
-Five-day scenario-based gameplay
-Energy and academic score tracking
-Interactive user choices
-Progress updates based on player decisions

## Technologies Used
-Java
-NetBeans IDE
-Object-Oriented Programming (OOP)

## Skills Demonstrated

-Problem Solving
-Control Structures and Loops
